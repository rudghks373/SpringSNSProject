## SpringSNSProject

전자정부프레임워크(스프링프레임워크)를 이용한 SNS 형식의 웹어플리케이션 토이프로젝트.

> 제작툴

- 전자정부프레임워크 3.7 version
- Tomcat 8.0
- JDK 8
